MultiRexZ80
===========

MultiRexZ80 is an emulator that can emulate several Z80-based platforms such as :
- SG-1000
- Sega Master System
- Sega Game Gear
- CBS Colecovision (with MegaROM support)
- Sord M5 (Only cartridges for now)
- Sega System E
- Sega System 1/2
- SNK Psycho Soldier-based
- SNK Triple Z80-based

it used to be called simply SMS Plus GX or Super SMS Plus GX but the scope has expanded so much (somewhat)
that it felt dishonest to claim it was "just" a Master System emulator, even if it remains the main focus of it.

It aims to be a performance and decently accurate emulator for the given systems with a low CPU usage and portability.
Cores can be selectively disabled/enabled (useful if you don't care for arcade games), and great care has been made to ensure
maximized performance.

See docs/contributors to see who contributed (indirectly or directly) to the previous SMS Plus (GX).

MultiRexZ80

Drop a Master System, Game Gear, SG-1000, Coleco, System E, or supported arcade ROM here.

Controls: arrows, Z/X, Enter/Pause. Arcade: 5 coin, 1 start, 9 service, F2 test. Block Gal dial uses left/right.

Build multirexz80.wasm with make -f Makefile.wasm, then serve this directory over HTTP.
